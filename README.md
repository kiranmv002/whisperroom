# WhisperRoom 🤫

A private real-time chat app where rooms self-destruct,
you can whisper secretly, and panic exit anytime.

> Built with pure HTML, CSS, and JavaScript.
> No installations. No frameworks. Just open and run.

---

## Features
- 🔐 6-digit private room codes
- 💬 Real-time chat
- 🤫 Whisper mode — private message one person inside a group room
- 😱 Panic button — instantly delete your trace and exit
- 👥 Live member list
- 🔒 Anti-copy + blur on tab switch
- 🛡️ Safe exit page after panic

---

## How to Run
1. Download or clone this repo
2. Open `html/index.html` in your browser
3. That's it — no installs needed!
